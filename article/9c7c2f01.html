教你怎么在手机上用命令行下载B站视频

发表于 2019-05-19 更新于 2020-08-05

为什么要用命令行下载视频？

因为又快又简单

前言

如果你浏览到一个视频，觉得很棒，想要下载下来，大家要做的肯定是打开自己的下载工具，将视频网址复制黏贴过去，然后点击确定下载。这样的下载软件有很多，比如IDM，比如迅雷，比如ADM，比如比特彗星，或者是其他的一些新兴的下载软件（我这里就不多介绍了，还想以后靠这些多写几篇推荐博客呢）。

今天我要在这里给大家再介绍一个用命令行操作的下载工具you-get,这个工具是GitHub上的一个项目，在电脑上的时候大家就已经用的很开心了。最近让我发现了将它搬到手机上用的方法，经过本人一下午跟着其他人的教程一通瞎捣鼓，最终成功可用，而且速度很快，操作简单。

相关文件的安装

首先下载手机端的终端

Termux # 在百度上搜索一下就有了

然后进入termux进行命令行操作下载相关文件

因为一些因为所以，科学道理的原理，termux要先配置一下才能用

  1. 更新 命令：pkg update $$ pkg upgrade # 先更新一下，以防你下载的不是最新版
  2. 请求权限 命令：termux-setup-storage # 这个是请求存储权限的，你要下载东西总得给他存储权限吧？
  3. 安装python 命令：pkg install python # 因为you-get工具是一个python程序，所以你要运行它需要相关的python`环境
  4. 安装ffmpeg 命令：pkg install ffmpeg # 用来采集视频的包
  5. 安装you-get 命令：pip3 install you-get

然后你就可以起飞了！！！

操作方法

工具装好了，那么该怎么用呢？

在termux上输入 you-get

当你进入you-get后，可以输入you-get来查看you-get的相关命令

举个例子，我们要下载B站的这个视频用来当动态桌面：

这个视频的网址是：

咬人猫跳舞视频

则我们需要用下面的命令：

you-get -o /storage/emulated/0/you-get "https://www.bilibili.com/video/av52727612/"

  1. you-get是调用you-get工具，

  2. -o /storage/emulated/0/you-get 是指定下载文件夹，

（这里的意思是我要把这个文件下载到/根目录下的storage文件夹中的emulated文件夹里的0文件夹下的you-get文件夹中）

  1. (”https://www.bilibili.com/video/av52727612/“ ) 是视频网址。
